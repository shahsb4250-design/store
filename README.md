# SMI Store

Premium Cyber Retail Store.

## Features

-   **Admin Control**: Secure admin portal (locked) to manage inventory.
-   **Inventory Management**: Add, delete, and manage product stock.
-   **Order processing**: View orders, customer details, and total amounts.
-   **Print Support**: Print individual order details or full manifests.
-   **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
-   **Reliable Backend**: Node.js/Express with MongoDB (or improved local fallback).

## Installation

1.  Navigate to the project directory.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the server:
    ```bash
    npm start
    ```
    OR for development:
    ```bash
    npm run dev
    ```
4.  Open `http://localhost:5000` in your browser.

## Tech Stack

-   **Frontend**: HTML5, Vanilla CSS, JavaScript.
-   **Backend**: Node.js, Express.js.
-   **Database**: MongoDB (Optional, falls back to in-memory).
